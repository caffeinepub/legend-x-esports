# Specification

## Summary
**Goal:** Replace the current PWA app icons with a new fire-and-ice dual tiger illustration.

**Planned changes:**
- Generate new PWA icons (192×192 and 512×512) featuring a split fire-and-ice dual tiger design inspired by the uploaded image
- Replace existing icon files in the PWA assets folder with the new icons
- Update `manifest.json` icon entries to point to the new icon files
- Update `index.html` favicon and apple-touch-icon references to use the new icons

**User-visible outcome:** The PWA app icon displayed on home screens and browser tabs will show the dramatic fire-and-ice dual tiger design.
